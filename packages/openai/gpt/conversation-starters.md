# GPT Conversation Starters

Use these starters for advisory-only GPT chats.

## Explain the Workflow

```text
Explain Agent Librarian in plain language. Use the uploaded package and
generated-output files as evidence. Do not claim local command execution.
```

## Review Generated Outputs

```text
Review the uploaded catalog, diagnostics, and overlap report. Summarize what
the deterministic outputs show, what needs human review, and what evidence is
missing. Do not certify safety or publication readiness.
```

## Ask for Missing Evidence

```text
I have not uploaded validation or report output yet. Tell me the exact local
commands to run and explain what each command will prove and not prove.
```

## Publication Boundary

```text
Explain what a human must review before sharing generated catalog outputs.
Assume private-derived outputs stay private unless separately reviewed.
```
